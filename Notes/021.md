#### 连续子数组的最大和
---
__题目描述__：
输入一个整数数组，有正有负。求所有子数组的和的最大值。例如:{1，-2，3,10，-4，7，2，-5}，最大子数组为{3，10，-4,7,2}，和为18

__思路__

1. 分析数据的规律，初始化和为0，加上数字1，sum=1，加上数字-2，sum=-1，后面的无论怎么加都是变小，sum<0就将下一个数赋给sum，注意全为负的情况
```python
class Solution:
    def FindGreatestSumOfSubArray(self, array):
        # write code here
        if array is None:
            return
        Sum = 0
        greatSum = array[0]
        for i in range(len(array)):
            if Sum<=0:
                Sum = array[i]
            else:
            	Sum += array[i]
            if greatSum<Sum:
                greatSum = Sum
        return greatSum
```

2. 动态规划，$f(i)=max(f(i-1)+arr[i],arr[i])$ 

```python
def FindGreatestSumOfSubArray(self, array):
        # write code here
        if array is None:
            return
        Sum = 0
        greatSum = array[0]
        for i in range(len(array)):
            Sum = max(Sum+array[i],array[i])
            if greatSum<Sum:
                greatSum = Sum
```

#### 构建乘积数组

---

**题目描述**

给定一个数组A[0,1,...,n-1],请构建一个数组B[0,1,...,n-1],其中B中的元素B[i]=A[0]*A[1]*...*A[i-1]*A[i+1]*...*A[n-1]。不能使用除法

**思路**

```python
class Solution:
    def multiply1(self, A):
        # write code here
        if not A:
            return []
        n = len(A)
        B = [None]*n
        B[0] = 1
        for i in range(1,n):
            B[i] = B[i-1]*A[i-1]    # B[i] = A[0]*...*A[i-1]
        tmp = 1
        for j in range(n-2,-1,-1):
            tmp *= A[j+1]
            B[j] *= tmp
        return B
     
    def multiply2(self, A):
        # write code here
        B = [1 for i in A]
        for i in range(len(A)):
            if i == 0 or i == len(A)-1:
                # lambda i:
                tempA = A[i+1:] if i==0 else A[:i]
            else:
                tempA = A[i+1:] + A[:i]
            for j in tempA:
                B[i] *= j
        return B
```
