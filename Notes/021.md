#### 连续子数组的最大和
---
__题目描述__：
输入一个整数数组，有正有负。求所有子数组的和的最大值。例如:{1，-2，3,10，-4，7，2，-5}，最大子数组为{3，10，-4,7,2}，和为18

__思路__

1. 分析数据的规律，初始化和为0，加上数字1，sum=1，加上数字-2，sum=-1，后面的无论怎么加都是变小，sum<0就将下一个数赋给sum，注意全为负的情况
```python
class Solution:
    def FindGreatestSumOfSubArray(self, array):
        # write code here
        if array is None:
            return
        Sum = 0
        greatSum = array[0]
        for i in range(len(array)):
            if Sum<=0:
                Sum = array[i]
            else:
            	Sum += array[i]
            if greatSum<Sum:
                greatSum = Sum
        return greatSum
```

2. 动态规划，$f(i)=max(f(i-1)+arr[i],arr[i])$ 

```python
def FindGreatestSumOfSubArray(self, array):
        # write code here
        if array is None:
            return
        Sum = 0
        greatSum = array[0]
        for i in range(len(array)):
            Sum = max(Sum+array[i],array[i])
            if greatSum<Sum:
                greatSum = Sum
```
